# Bitcoin MACD & Elliott Wave Chart

An interactive Bitcoin price chart with technical analysis indicators, featuring MACD and Elliott Wave annotations.

## Features

- **Interactive Bitcoin Price Chart** - Built with Plotly.js for smooth zooming, panning, and hover interactions
- **MACD Indicator** - Moving Average Convergence Divergence with standard parameters (12, 26, 9)
- **Elliott Wave Annotations** - Visual wave pattern markers on the chart
- **Real-time Data** - Fetches Bitcoin price data from CoinGecko API (free, no API key required)

## Usage

### View Online

Visit the deployed GitHub Pages site to view the chart.

### Deploy Your Own

1. Fork this repository
2. Enable GitHub Pages in repository settings
3. Set the source to deploy from the `main` branch
4. Access your chart at `https://<username>.github.io/<repo-name>/PAGE/BTC-elliott-wave-illucidations/`

## Technologies

- HTML5
- JavaScript (ES6+)
- [Plotly.js](https://plotly.com/javascript/) - Interactive charting library
- [CoinGecko API](https://www.coingecko.com/en/api) - Free cryptocurrency data
